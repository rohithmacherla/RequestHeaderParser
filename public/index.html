Request Header Parser Microservice

Purpose: This API will simply output some meta data about the user such as their IP address, language of choice, and browser/software that he or she is running to view this page (all in JSON).

Example call:

https://quiet-badlands-14995.herokuapp.com/whoami

Example JSON output:

{"ip":"::ffff:10.150.98.212", "language":"en-US", "browser":{"isAuthoritative":true,"isMobile":false,"isTablet":false, "isiPad":false,"isiPod":false,"isiPhone":false, "isAndroid":false,"isBlackberry":false ...}}
